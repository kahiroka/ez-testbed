# ez-testbed
Signal generation and probing testbed for Arduino

# Pin Assign
| Pin   | Function                   |
| ----- | -------------------------- |
| D0-7  | 8-bit counter              |
| A8-11 | 1-hot OUTPUT/INPUT_PULLUPs |
| D12   | Not used                   |
| D13   | Not used (Builtin LED)     |
| A0-5  | Probe for Serial Plotter   |

# Usage
Use this with a breadboard for testing transistor or something.
